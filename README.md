## 게임 컨셉
  모바일 리듬게임
  4key / 6key
  레벨 편집기


## 개발 범위
  일반 노트, 롱 노트, 플릭 노트, 슬라이드 노트
  
  커스텀 레벨 편집기 구현하기: 유저가 음악에 맞추어 key 입력하면 유저의 입력을 바탕으로 패턴이 입력/저장됨. 저장된 패턴으로 곡 플레이 가능
  
  + 입력한 노트 타이밍/박자 자동 보정 및 세부 수정
    
  + 커스텀 맵 파일로 저장/불러오기

## 개발 일정
  1주차: 프레임워크 정리 (현재 수업 + 이전 수업 내용 참고)
  
  2주차: 프레임워크 제작 및 수정: 시스템, ui
  
  3주차: 음원 불러오기, 일반노트(탭노트) 패턴 구현해보기 
  
  4주차: 레벨 편집기 기초 구현: 노래에 맞추어 key 입력하면 유저가 입력한대로 패턴생성
  
  5주차: 롱노트/롱노트 패턴 생성 구현하기  ++ 박자 보정 구현
  
  6주차: 플릭노트(짧게 슬라이드), 슬라이드 노트(길게 누른채로 슬라이드) 구현
  
  7주차: 기본 맵 제작, 디자인 보완  ++ 맵을 파일로 저장/불러오기 구현
  
  8주차: 부족한 부분 수정/보완, 최종 테스트
