# Draw The Beat

## 게임 컨셉
  모바일 리듬게임
  4key / 6key
  레벨 편집기


## 개발 범위
  일반 노트, 롱 노트, 플릭 노트, 슬라이드 노트
  
  커스텀 레벨 편집기 구현하기: 유저가 음악에 맞추어 key 입력하면 유저의 입력을 바탕으로 패턴이 입력/저장됨. 저장된 패턴으로 곡 플레이 가능
  
  \+ 입력한 노트 타이밍/박자 자동 보정 및 세부 수정
    
  \+ 커스텀 맵 파일로 저장/불러오기

## 개발 일정
  1주차: 프레임워크 정리 (현재 수업 + 이전 수업 내용 참고)
  
  2주차: 프레임워크 제작 및 수정: 시스템, ui
  
  3주차: 음원 불러오기, 일반노트(탭노트) 패턴 구현해보기 
  
  4주차: 레벨 편집기 기초 구현: 노래에 맞추어 key 입력하면 유저가 입력한대로 패턴생성
  
  5주차: 롱노트/롱노트 패턴 생성 구현하기  \+ 박자 보정 구현
  
  6주차: 플릭노트(짧게 슬라이드), 슬라이드 노트(길게 누른채로 슬라이드) 구현
  
  7주차: 기본 맵 제작, 디자인 보완  \+ 맵을 파일로 저장/불러오기 구현
  
  8주차: 부족한 부분 수정/보완, 최종 테스트
  
## 개발 진행
    2주차: 음원재생, audioClock, Note 클래스 정의, lane 설정
    3주차: 노트 입력 저장, json 파일 생성 및 불러오기 구현, 박자 스냅
    6주차: 노트 중복 생성 해결, 박자 스냅 수정 
    7주차: Scene 세부 수정 타이틀/홈/플레이/에디트, 터치 판정/타이밍 표시, 효과음 추가
    8주차: hp, 점수 계산, 클리어 화면, 음원/효과음 음량 설정, 맵 제작 - 안드로이드 기기에서 제작

## 커밋
    커밋이 진행되지 않음
    2,3주 프레임워크 제작 이후 
    7~8주차에 몰아서 작업했다

## 사용된 기술
    MediaPlayer의 재생 시간을 audioclock을 구현하여 음원과 채보의 재생 타임을 계산(MediaPlayer.currentPosition 확인)하고, 음원 시각과 입력한 노트 시각을 동기화하여 노트 입력, 노트 판정을 가능하게 한다.
    ->터치한 순간의 audioclock을 저장
    snap 값을 설정해 설정된 bpm의 1/4 1/6 1/8 1/16등의 박자로 자동으로 보정한다.
    음원을 재생하며 만든 맵을 JSON 파일로 맵 데이터 내보낸다. JSON파일에 저장된 맵의 노트 정보를 불러와 맵을 플레이 할 수 있다.

## 아쉬운 점
    이펙트나 애니메이션이 부족하고 화면 UI가 부실하다.
    플릭과 슬라이드 노트는 구현하지 못하였다.
    노래나 맵파일을 외부에서 가져올 수 없다.
    곡 선택 화면에서도 썸네일이나 다른 이미지를 사용하지 않아 밋밋하다.
    여러개의 레인 입력을 동시에 저장할 수 없어 되감기를 해서 입력해야한다.

## 수업에 대한 내용
    수업을 통해 kotlin 사용한 안드로이드앱 개발에 대해 배울 수 있어 좋았다.
    커밋을 하나씩 따라가며 의도를 분석하며 수업을 따라갈 수 있었다.
    안드로이드와 Kotlin 뿐만이 아닌 게임개발에 사용되는 여러 스킬들을 배우고 생각해볼 수 있었다.
